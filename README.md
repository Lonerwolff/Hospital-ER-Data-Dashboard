# Hospital Emergency Room Dashboard (Excel)

## Overview
This project is an interactive Hospital Emergency Room Dashboard built in Microsoft Excel. The dashboard provides a comprehensive view of emergency room performance, patient demographics, admission status, wait times, satisfaction scores, and department referrals. It helps healthcare administrators monitor key performance indicators (KPIs) and make data-driven decisions to improve patient care and operational efficiency.

## Objective
The primary objective of this dashboard is to analyze emergency room data and provide actionable insights regarding patient flow, service quality, and resource utilization.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Visualization

## Key Performance Indicators (KPIs)
- Total Number of Patients
- Average Wait Time
- Patient Satisfaction Score
- Admission vs Non-Admission Rate
- Patient Attendance Within Time
- Gender Distribution
- Department Referral Analysis
- Age Group Analysis

## Dashboard Features

### Monthly Filtering
- Interactive month-wise selection using slicers.
- Dynamic dashboard updates based on selected month.

### Year Selection
- Compare and analyze data across multiple years.

### Patient Analysis
- Total patients visiting the emergency room.
- Admission and non-admission breakdown.
- Gender-wise patient distribution.
- Age-group-wise patient analysis.

### Operational Performance
- Average patient wait time.
- Percentage of patients attended on time.
- Patient satisfaction monitoring.

### Department Referral Analysis
- Track referrals across departments such as:
  - General Practice
  - Orthopedics
  - Cardiology
  - Physiotherapy
  - Gastroenterology
  - Neurology
  - Renal

## Business Insights
- Identify peak patient load periods.
- Monitor emergency room efficiency.
- Evaluate patient satisfaction trends.
- Analyze referral patterns across departments.
- Support hospital management in resource planning and decision-making.
